\doxysubsection{Detailed Description}
AERN Help and Guidance Functions. 

This header declares functions that provide context-\/sensitive help messages for the AERN console interface. These functions print help strings that explain available command actions and options based on the current console mode (e.\+g., configuration, enable, or user mode) and the device\textquotesingle{}s network designation (e.\+g., APS, Client, ADC, ARS). The help functions make use of pre-\/defined help text strings (stored in the AERN\+\_\+\+APPLICATION\+\_\+\+HELP\+\_\+\+STRINGS array) and output them using the QSC console utilities.

The primary functions provided in this module are\+:


\begin{DoxyItemize}
\item aern\+\_\+help\+\_\+print\+\_\+context()\+: Prints a single help message associated with a specific command action.
\item aern\+\_\+help\+\_\+print\+\_\+mode()\+: Prints a set of help messages tailored to the current console mode and network designation.
\end{DoxyItemize}

These functions are essential for providing users with guidance on how to interact with the AERN system.
