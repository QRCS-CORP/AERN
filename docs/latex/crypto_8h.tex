\doxysubsection{Detailed Description}
AERN Cryptographic Functions. 

This header defines the cryptographic functions used by the Anonymous Encrypted Relay Network (AERN). The crypto module encapsulates all operations required for secure data processing including\+:


\begin{DoxyItemize}
\item Stream encryption and decryption using a symmetric cipher based on the RCS (Randomized Cipher Stream) algorithm.
\item Generation of secure key chains by combining a user\textquotesingle{}s password and username with a unique application salt.
\item Generation of message hashes and message authentication codes (MACs) using SHA3-\/256 and KMAC256.
\item Password handling including a minimum quality check, secure password hashing, and verification.
\item Secure memory management functions to allocate and deallocate memory that is immediately cleared to protect sensitive data.
\end{DoxyItemize}

Two configuration macros are provided to tune the passphrase hashing function\+:
\begin{DoxyItemize}
\item \doxylink{crypto_8h_a05c3c8bda8ff3b3a231a3c82bc648120}{AERN\+\_\+\+CRYPTO\+\_\+\+PHASH\+\_\+\+CPU\+\_\+\+COST} sets the CPU cost in iterations.
\item \doxylink{crypto_8h_aa602823c0146ee7cce254c4343acd8f8}{AERN\+\_\+\+CRYPTO\+\_\+\+PHASH\+\_\+\+MEMORY\+\_\+\+COST} sets the memory cost in megabytes.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
These cryptographic operations build upon underlying primitives provided by the QSC library (e.\+g. SHA3, c\+SHAKE, KMAC, and RCS). Correct operation of these functions is critical to the security of the AERN protocol.
\end{DoxyNote}
\begin{DoxyRefDesc}{Test}
\item[\mbox{\hyperlink{test__test000003}{Test}}]The AERN crypto functions are rigorously tested to ensure that\+:\end{DoxyRefDesc}



\begin{DoxyItemize}
\item Data streams are properly encrypted and decrypted.
\item The application key chain is securely generated from a user\textquotesingle{}s password and username, using a salt derived from OS sources.
\item Message hashes and MAC codes are computed accurately.
\item Passwords are validated against a set minimum complexity and correctly verified against stored hashes.
\item Secure memory is allocated and deallocated without leaving residual data.
\end{DoxyItemize}

These tests help ensure that the cryptographic foundation of AERN is robust and reliable.
