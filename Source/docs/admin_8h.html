AERN_EXPORT_API void aern_background_start ( aern_background_state * bg)

Start the background cover-traffic thread.

Spawns a thread that calls the background traffic loop. Must call aern_background_stop() before process exit.

Parameters
bg[aern_background_state*] Background state; running is set to true.

◆ aern_background_thread()

AERN_EXPORT_API void aern_background_thread ( void * arg)

Background cover-traffic thread function.

Every AERN_BACKGROUND_INTERVAL_MS, sends one random-padded 1500-byte encrypted packet on each active cipher-table entry. Thread exits when bg->running == false.

Parameters
arg[void*] Pointer to aern_background_state.

AERN_EXPORT_API aern_protocol_errors aern_log_write ( aern_log_state * ls,
const char * devname,
aern_protocol_errors error )

Append one entry to the encrypted ADC log.

The entry is formatted, appended to the plaintext, and the entire file is re-encrypted. This is consistent with the pattern in server.c.

Parameters
ls[aern_log_state*] Open log state.
devname[const char*] Device name string (up to AERN_LOG_DEVICE_NAME_SIZE bytes).
error[aern_protocol_errors] Protocol error code.
Returns
Returns aern_protocol_error_none on success.
