Detailed Description

Deprecated MPDC-derived fragment-control message compatibility API.

AERN does not use MPDC-style multi-party key-fragment strengthening. The fragment-collection and fragment-query control messages are therefore not part of the active AERN protocol path. Relay payload fragmentation, out-of-order fragment caching, and reassembly are implemented in route.c.

These declarations are retained to satisfy legacy references while obsolete fragment-control flags are being removed from the service dispatch path. The implementations reject all calls with a protocol error and do not transmit fragment-control packets.
