Detailed Description

Encrypted relay payload header.

This header is serialized inside the encrypted relay payload. Intermediate APS nodes cannot read it unless they are processing the packet as the current decrypting hop. The terminal egress uses this header to associate opaque serialized packets and fragments with a previously established session cache entry.
