◆ aern_adc_alive_poll_thread()

AERN_EXPORT_API void aern_adc_alive_poll_thread ( void * arg)

Background thread: ADC polls each APS every AERN_NODE_ALIVE_POLL_S seconds.

  • Sends { adc_cert || timestamp || adc_sig } to each APS.
  • Expects a response within AERN_NODE_ALIVE_TIMEOUT_S seconds from last response.
  • On timeout: marks inactive, calls aern_network_revoke_broadcast, logs.
Parameters
arg[void*] Pointer to aern_adc_alive_poll_state.

Forward-path transmit failure: attempt hop bypass.

Parameters
fwd[aern_forward_state*] Forward state.
plaintext[uint8_t*] Decrypted plaintext (1446 bytes) to re-route.
rm[aern_route_map*] Route map whose future-hop slots may be consumed.
failed_addr[const char*] IP of the unresponsive hop.
entry_addr[const char*] Entry node IP (for error report).
Returns
Returns aern_protocol_error_none if bypass succeeded.
