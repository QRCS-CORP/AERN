\doxysubsection{Detailed Description}
Deprecated MPDC-\/derived fragment-\/control message compatibility layer. 

AERN does not use the MPDC multi-\/party fragment-\/strengthening protocol. Relay payload fragmentation and out-\/of-\/order reassembly are implemented by the route layer. The functions in this file are retained only to satisfy existing link references and to reject obsolete network control flags safely.
