\doxysubsection{Detailed Description}
Relay pending and delay queue interface. 

This header defines the bounded relay queue used by the AERN route engine for packets that are waiting for session establishment or delayed ingress release. The queue stores a fixed number of item slots, while each queued packet is held in heap-\/backed storage owned by the queue entry. The queue does not use qsc\+\_\+queue\+\_\+state and does not expose internal ring indexes to callers.

Queue ownership rules\+:
\begin{DoxyItemize}
\item Push copies the caller-\/provided packet into queue-\/owned storage.
\item Pop transfers the queue-\/owned packet storage to the caller-\/owned item.
\item Remove, cleanup, clear, and dispose securely erase and release all owned packet storage before clearing queue state.
\end{DoxyItemize}

Callers that receive an item through a successful pop operation must release the transferred packet storage with \doxylink{relayqueue_8h_ab5f7f9826891efbb517b35cb57d268ec}{aern\+\_\+relayqueue\+\_\+item\+\_\+dispose()}.
