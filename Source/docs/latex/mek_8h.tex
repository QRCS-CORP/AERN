\doxysubsection{Detailed Description}
Master Encryption Key (MEK) derivation, APS cipher-\/table management, and proxy mesh orchestration. 

The wire exchange is performed by the canonical MFK exchange functions in network.\+c. This module converts the resulting shared secret into directional RCS send and receive states, stores peer tunnel state, and tracks the MEK rekey threshold for APS-\/to-\/\+APS tunnels.

On success\+: add connection state to the cipher table. f. On failure\+: mark the peer unsynchronised and continue.
