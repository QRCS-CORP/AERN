\doxysubsection{Detailed Description}
Wraps a topology list with a monotonically increasing version counter. 

The version starts at 1 after initialisation and is incremented every time the topology is structurally modified (add, update, or remove). Version 0 is reserved as "{}uninitialised"{}; any peer sending version 0 is rejected.
