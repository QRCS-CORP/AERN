AERN ARS root initialization and lifecycle tests. 

This header declares the test-\/group entry point for ARS root certificate generation, retained signing-\/key use, child certificate signing, invalid generation parameters, and signing-\/key unload behavior. The declarations are test-\/harness entry points and are not part of the AERN production API.
