Route-\/layer validation tests for the AERN relay path. 

This header declares the route-\/layer test cases used by AERNTest. The tests cover route-\/map encoding, route generation, relay payload-\/header encoding, session-\/control encoding, relay-\/queue cleanup, fragment ownership, and dynamic fragment reassembly behavior.
