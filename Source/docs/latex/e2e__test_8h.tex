\doxysubsection{Detailed Description}
Deterministic end-\/to-\/end substrate tests for AERNTest. 

This header declares the deterministic end-\/to-\/end test group used by the AERNTest harness. The tests verify local relay, session, callback, return-\/path, dummy-\/traffic, and fragment-\/reassembly boundaries that can be exercised without launching interactive ARS, ADC, APS, or ACD console processes.

\begin{DoxyNote}{Note}
Live socket and multi-\/process orchestration remain outside this local deterministic test group. 
\end{DoxyNote}
