\doxysubsection{Detailed Description}
The ARS remote signing handler state. 

State for the ARS-\/side signing handler.

This structure binds the accepted ADC socket, the cached ADC certificate, the ARS root certificate, and the ARS private signing key used to process an authenticated remote certificate signing request.
