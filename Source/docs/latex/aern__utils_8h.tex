\doxysubsection{Detailed Description}
Common test harness utilities for AERNTest. 

This header defines the shared output, conversion, and case-\/dispatch helpers used by the AERN deterministic test harness. The declarations are test-\/only utilities and are not part of the AERN protocol API.
