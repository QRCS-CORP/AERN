\doxysubsection{Detailed Description}
AERN Anonymous Proxy Server (APS). 

The APS is the workhorse of the AERN network. Each APS holds a child certificate signed by the domain\textquotesingle{}s ARS and registered with the ADC. APS nodes relay encrypted client traffic, participate in topology convergence, exchange master fragment keys with peers, and forward registration requests from clients to the ADC.
